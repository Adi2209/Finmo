## Description

Forex Trading System - Backend Engineer Assignment 

## Installation

```bash
$ npm install
```
---
## Running the app

#### Step 1: Create a .env file in the root directory of the project and populate it with the .env.example file 

#### Step 2: Save the .env file.

<br>

```bash
# development
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
